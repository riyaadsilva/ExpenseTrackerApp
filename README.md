The Expense Tracker Application is a JavaFX-based system designed to assist users in systematically recording, managing, and analyzing their daily financial activities. The application enables users to add, update, and categorize expenses while providing a structured interface for monitoring spending patterns over time. To enhance accuracy in financial computation, the system integrates a real-time currency exchange service that ensures reliable multi-currency support. The application architecture combines an intuitive user interface with robust data processing logic, thereby offering a comprehensive and user-friendly solution for personal financial management. Overall, the system aims to promote better budgeting practices and informed financial decision-making through efficient tracking and visualization of expenses.
